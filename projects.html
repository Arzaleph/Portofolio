Proyek Saya

Sistem Informasi Perpustakaan

Aplikasi berbasis web untuk mengelola data buku, anggota, serta peminjaman dan pengembalian. Dibangun menggunakan PHP dan MySQL, dengan antarmuka yang mudah digunakan untuk admin maupun pengguna.

Aplikasi Pemesanan Tiket Konser Online

Platform pemesanan tiket konser berbasis web yang mendukung pemilihan kursi, pembayaran online, serta notifikasi email. Dibuat sebagai bagian dari tugas besar mata kuliah Rekayasa Perangkat Lunak.

Implementasi IoT untuk Monitoring Suhu & Kelembaban

Proyek IoT menggunakan sensor DHT22 dan ESP8266 untuk mengukur suhu dan kelembaban secara real-time, dengan data ditampilkan melalui dashboard berbasis web menggunakan Node-RED.
